A B C

(The Accessible Books Consortium) El Consorcio de Libros Accesibles tiene como objetivo aumentar el número de libros en todo el mundo en formatos accesibles y ponerlos a disposición de las personas ciegas, con problemas de visión o con problemas de lectura. ABC es una asociación de múltiples partes interesadas, que comprende a la OMPI, organizaciones que sirven a personas con problemas de lectura y organizaciones que representan a editores y autores.

Acceso abierto

"El acceso abierto (en inglés, Open Access, OA) es el acceso gratuito a la información y al uso sin restricciones de los recursos digitales por parte de todas las personas. Cualquier tipo de contenido digital puede estar publicado en acceso abierto: desde textos y bases de datos hasta software y soportes de audio, vídeo y multimedia. A pesar de que la mayoría del contenido digital disponible está constituido exclusivamente por texto, un número cada vez mayor de recursos combina textos con imágenes, bases de datos y archivos ejecutables. El acceso abierto también puede aplicarse a contenido no académico como música, películas y novelas." (UNESCO, n.d.).

AMIS reader

AMIS reader: AMIS es un acrónimo de Adaptive Multimedia Information System. AMIS, una herramienta de software gratuita y de código abierto para reproducir libros DAISY 2.02 y DAISY 3-2005, ya no se mantiene, pero está disponible para referencia y uso en sistemas más antiguos. Se puede encontrar más información en el proyecto archivado de AMIS.

Android:es un sistema operativo basado en Linux. Fue diseñado para dispositivos móviles con pantalla táctil, como teléfonos, tabletas, relojes inteligentes, televisores y autos.

API: (application programming interface) Interfaz de programación de aplicaciones.

ASCII: (American Standard Code for Information Interchange) Código Estándar Americano para Intercambio de Información. Consulte ASCII format for Network Interchange.

Audio NCX: Se refiere a un libro DAISY que contiene el libro completo como audio sincronizado, más una tabla de contenido (que podría incluir números de página, listas de notas al pie, etc.). Este término no se refiere a una versión específica del estándar DAISY.

Audiolibro: Una grabación de una lectura de voz humana o una interpretación de voz sintética de un libro.

Biblioteca en Acción: El programa radial "Biblioteca en Acción", creado en el año 2020, tiene como fin incentivar la lectura y la escritura, entregando toda la información en torno a los libros que le interesan a los oyentes y seguidores.

Colecciones: es la categoría donde se almacenan los ítems, en esta no es posible albergar otras colecciones y debe pertenecer a una comunidad o subcomunidad.

Comunidades: Categoría más amplia en Dspace, dentro de ellas pueden estar contenidas comunidades o colecciones, pero no puede contener directamente un ítem.

DAISY: El formato DAISY es un formato de libro digital que está diseñado para ser un sustituto de audio completo del material impreso y está diseñado específicamente para que lo utilicen personas con discapacidades para la lectura, como ceguera, problemas de visión y dislexia. Basado en los formatos MP3 y XML, el formato DAISY tiene funciones de navegación avanzadas además de las de un audio libro tradicional.

Datasets: Conjunto de datos que son parte del trabajo de una investigación.

Document type declaration: En documentos válidos, la declaración que conecta un documento con su definición de tipo de documento. La declaración puede conectarse a un archivo externo o incluir la definición dentro de sí misma.

Documento estructurado accesible: es un documento que conserva una organización lógica que permite navegar fácilmente dentro del contenido del archivo, adicionalmente contiene un marcado de idioma y orden de lectura adecuado. Están orientados hacía las personas con discapacidad visual.

DOCX: es un formato muy conocido para documentos de Microsoft Word. Introducido a partir de 2007 con el lanzamiento de Microsoft Office 2007, la estructura de este nuevo formato de documento se cambió de binario simple a una combinación de XML y archivos binarios. Los archivos Docx se pueden abrir con Word 2007 y versiones laterales, pero no con las versiones anteriores de MS Word que admiten extensiones de archivo DOC.

DRM (Digital Rights Management): se refiere a las medidas tecnológicas aplicadas a los contenidos digitales que otorgan derechos específicos. DRM se refiere en términos generales a cualquier estrategia o herramienta que se utiliza para controlar quién puede acceder al contenido digital y cómo puede usarlo. DRM puede ser tan simple como poner un aviso de derechos de autor en un documento, o puede ir mucho más lejos y encriptar un documento con un código especial o clave requerida para acceder al contenido. El cifrado es un ejemplo de medida de protección tecnológica (TPM), que es una especie de DRM, pero los términos a menudo se usan indistintamente.

DSpace: Software de código abierto usado generalmente para gestionar repositorios instituciones, permite almacenar todo tipo de formatos digitales, facilitando su organización, descripción y recuperación del contenido.

EPUB: abreviatura de publicación electrónica, es un estándar de libros electrónicos gratuito y abierto.

IOS: es un sistema operativo móvil de la multinacional Apple Inc, desarrollado para iPhone.

Ítem: Un ítem esta compuesto por una serie e metadatos y un conjunto de archivos.

Libro electrónico: publicación electrónica o un libro electrónico, generalmente (pero no necesariamente) una versión digital de una publicación impresa. Hay muchos formatos de libros electrónicos.

Licencia de uso: Autorización por la cual el autor de una obra indica qué usos permite ejercer sobre ella a terceros. Existen diferentes tipos de licencias de uso: desde la más restrictiva ("todos los derechos reservados") hasta las permiten la cesión parcial de los derechos de explotación bajo determinadas condiciones / que especifican una variedad de usos posibles bajo determinadas condiciones (es el caso de las Licencias Creative Commons).

Licencias Creative Commons: son modelos de contratos que sirven para otorgar públicamente el derecho de utilizar una publicación protegida por los derechos de autor.

Materia: Asunto, palabra clave, tema o frase que describe al recurso. Es un punto de acceso que tienen los recursos para ser consultados.

Metadatos: Un conjunto de datos que describe y proporciona información sobre otros datos, literalmente datos sobre datos. Los metadatos describen e identifican una publicación con fines de búsqueda y recuperación (hallazgo), indexación y archivo. El usuario final no es necesariamente consciente de su presencia.

MP3: es la extensión de archivo para MPEG, capa de audio 3. La capa 3 es uno de los tres esquemas de codificación MPEG (capa 1, capa 2 y capa 3) para la compresión de señales de audio. La capa 3 utiliza codificación de audio perceptual y compresión psicoacústica para eliminar toda la información superflua (más específicamente, las partes redundantes e irrelevantes de una señal de sonido que el oído humano no escucha). También agrega una MDCT (Transformada de coseno discreta modificada) que implementa un banco de filtros, aumentando la resolución de frecuencia 18 veces mayor que la de la capa 2.

NVDA (Non Visual Desktop Access): es un lector de pantalla libre y gratuito desarrollado por NV Access que permite a las personas ciegas y con discapacidad visual usar ordenadores. Para ello lee el texto que se muestra en pantalla mediante una voz sintética. Se puede controlar lo que NVDA lee moviendo el cursor al área relevante que contiene el texto, tanto poniendo el ratón encima como usando las flechas del teclado. NVDA también puede convertir el texto en braille si el usuario del ordenador posee un dispositivo llamado «pantalla braille». NVDA abre la puerta de la educación y el empleo a muchas personas ciegas. También da acceso a las redes sociales, la banca, compras en línea y noticias. (Comunidad de NVDA, n.d.).

PDF: "Formato de documento portátil (más conocido por la sigla PDF), para presentar e intercambiar documentos de manera fiable, independientemente del software, el hardware o el sistema operativo. El PDF es ahora un estándar abierto, reconocido por la Organización Internacional para la Estandarización (ISO)." (Adobe, n.d.).

Repositorio institucional: "Conjunto de servicios Web centralizados, creados para organizar, gestionar, preservar y ofrecer acceso libre a la producción científica, académica o de cualquier otra naturaleza cultural, en soporte digital, generada por los miembros de una institución." (Cano Inclán et al., 2015).

RTF: Rich Text Format, un formato de archivo de documento patentado desarrollado por Microsoft para el intercambio de documentos entre plataformas.

Subcomunidades: es una subcategoría dentro de una comunidad, puede contener subcomunidades o colecciones, pero no un ítem directamente.

Thorium Reader: es el lector de EPUB elegido para Windows 10, MacOS y Linux. Esta aplicación EDRLab está en constante desarrollo y pretende convertirse en una referencia para acceder a publicaciones EPUB 3 en formato reflow o diseño fijo, audiolibros y narrativas visuales, documentos PDF y libros electrónicos accesibles DAISY 2.02 y 3; LCP protegido o no. (Thorium Reader – EDRLab, n.d.).

Thumbnail: Conjunto de archivos miniatura de imágenes pertenecientes al registro original.

TTS: Texto a voz por sus siglas en inglés "text-to-speech".

URI: Uniform Resource Identifier: Un identificador uniforme de recursos es una cadena compacta de caracteres para identificar recursos: documentos, imágenes, archivos de audio, etc. Dentro de una DTB, es más probable que los URI aparezcan como valores de atributo para varios elementos XML, utilizados como una forma de identificar otros documentos o archivos ya sea en su totalidad o en parte. Un URI puede incluir un identificador de fragmento.

URL: Uniform Resource Locator: Un localizador uniforme de recursos (URL) es un subconjunto del identificador uniforme de recursos (URI) que especifica dónde está disponible un recurso identificado y el mecanismo para recuperarlo. En el uso popular y en muchos documentos técnicos y discusiones verbales, a menudo se usa incorrectamente como sinónimo de URI.
